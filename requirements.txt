# Dependencias del Sistema de Gestión Comercial
# Este archivo lista las dependencias necesarias para ejecutar la aplicación

# Dependencias básicas (ya incluidas en Python estándar)
# tkinter - GUI (incluido en Python estándar)
# sqlite3 - Base de datos (incluido en Python estándar)

# Dependencias opcionales para desarrollo
# pyinstaller - Solo para crear ejecutables .exe
# winshell - Solo para crear accesos directos en Windows (opcional)

# Para crear ejecutables .exe:
# pip install pyinstaller

# Para crear accesos directos automáticamente en Windows:
# pip install winshell pywin32

# Nota: La aplicación está diseñada para funcionar con Python 3.7+
# sin dependencias externas adicionales
